About Espate

EspateOS is an OS for creating distributions with PinguyBuilder.

This slide show can also be easily edited in HTML. (Please refer to the documentation in EspeOS for details.)

EspateOS has a document that describes how to customize it easily.

EspateOS is based on UbuntuMini.

In addition, OS which SereneLinux author created long ago will be created based on EspeateOS in the future.

It can be customized only by replacing the file contents, and some configuration files are optimized for distribution.

EspateOS comes standard with a lightweight, easily customizable Xfce.
