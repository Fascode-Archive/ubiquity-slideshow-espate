About Espate

EspateOS is an OS for creating distributions with PinguyBuilder.

This slide show can also be easily edited in HTML. (For details, please see the EspeOS wiki.)

EspateOS is based on UbuntuMini and is also based on SereneLinux.

It can be customized only by replacing the file contents, and some configuration files are optimized for distribution.

EspateOSは標準では軽量でカスタマイズしやすいXfceを搭載しています。
